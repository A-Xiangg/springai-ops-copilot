# SpringAI Ops Copilot

一个基于 Spring AI 构建的企业知识库与故障排查智能助手平台。

## Tech Stack
- Java 17
- Spring Boot 3.5.x
- Spring AI 1.1.x
- PostgreSQL + pgvector
- Redis
- Vue 3 + TypeScript + Ant Design Vue
